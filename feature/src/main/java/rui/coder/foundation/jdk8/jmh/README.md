# JMH (the java Microbenchmark Harness )java 性能基准测试


通过 集成jar 包启动

``` xml

<!--the Java Microbenchmark Harness ** java 性能基准测试 治理组件-->
<dependency>
    <groupId>org.openjdk.jmh</groupId>
    <artifactId>jmh-generator-annprocess</artifactId>
    <version>1.20</version>
</dependency>

```
